## XP-470B (máy cuộn 4″)
- Spec: 203 DPI, max print width **108mm**, media 20–118mm — **không** phải máy 80mm hẹp.
- In đúng: `SIZE 100 mm,80 mm`, trang = tem, **rotation 0**.
- Mode mặc định `xp470b`; giữ `narrow80` cho máy print-head ~80mm.
