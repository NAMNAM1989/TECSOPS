## PDF/In lưu cả trang web + tên file không theo AWB

**Triệu chứng:** Bấm lưu file → PDF = shell TCS (Hotline/GIỚI THIỆU), không phải phiếu; tên file không theo AWB.

**Nguyên nhân:**
1. Capture/print sớm trên trang chính thay vì iframe `about:blank` (phiếu «SHIPPER'S INSTRUCTION…»).
2. Đường **In ESID** không set `document.title` theo AWB (`suggest` chỉ khi `for_save_pdf`).
3. Marker nhận phiếu quá yếu (`esid`/`awb` có trên list UI) → nhận nhầm shell.

**Fix (`esid_page.py` + `tcs_client.py`):**
- Marker mạnh `_ESID_BILL_MARKERS`; cấm `page.pdf` shell.
- DOWNLOAD: chờ iframe/popup phiếu, PDF từ HTML frame, title AWB.
- PRINT: chặn print shell → chờ phiếu → set title `{AWB}_ESID` → `print()` trên frame phiếu.
