## ESID: xem form + HOÀN TẤT từ Ops (máy khác)

- Điền chạy trên agent → không mở được tab TCS cá nhân đã điền.
- Sau `declare-fill`: screenshot → `preview_file` / `GET /docs?file=` (PNG được phép).
- `POST /esid/declare-submit` + `confirm_submit: true`: verify AWB form → tick agree → HOÀN TẤT.
- Ops: panel ảnh preview + nút «HOÀN TẤT trên TCS» (confirm) trên thanh TCS.
- Điền vẫn mặc định không tự submit.

## ESID headed máy kho (hướng 1)

- Local/`npm run dev`/`tcs:agent:real`: mặc định `TCS_HEADLESS=0` — Chrome thật.
- Railway `start-fullstack`: mặc định `TCS_HEADLESS=1`.
- Sau Điền: `focus_window` + Ops nút «Hiện Chrome»; kiểm tra trên máy kho rồi HOÀN TẤT Chrome hoặc Ops.
- `POST /session/focus`; health trả `headless`.

## Login hiện Chrome TCS thật

- `POST /session/open` + `visible:true` (nút Login Ops): force headed, hiện ESID/Agent, focus cửa sổ.
- Quét/Điền/Submit/Prepare/Job: `focus_if_headed` trước thao tác.
- Launch: thử Chrome channel → Chromium bundled (Railway không có `/opt/google/chrome`).
- Railway: headed fail → fallback headless + ảnh `TCS_LOGIN_PREVIEW_*.png` trên Ops.
- Live view Ops: `GET /session/screenshot` (TCS_LIVE_VIEW.png) + panel «Xem live» poll ~2s — kiểm tra thao tác khi không có cửa sổ Chrome.

## noVNC TCS desktop (Railway)

- Docker: `xvfb` + `fluxbox` + `x11vnc` + `novnc`/`websockify`; script `scripts/start-tcs-desktop.mjs`.
- `start-fullstack`: `TCS_VNC=1` → desktop rồi agent `DISPLAY=:99` `TCS_HEADLESS=0`; `TCS_VNC=0` → headless.
- Express `server/tcsDesktopProxy.mjs`: `/tcs-desktop` HTTP+WS → `127.0.0.1:6080`.
- Ops nút **TCS desktop** → noVNC; «Xem ảnh» = screenshot phụ.
- Railway: `TCS_VNC_PASSWORD` tùy chọn (trống = `-nopw`); volume `TCS_BROWSER_PROFILE`.
